Password Generator

Description:
This is a command-line password generator made in Node.js. This program allows the user to generate random passwords with customizable lengths and the option to include or exclude special characters.

Installation
1. Clone the repository.
2. Run `npm install` to install dependencies (if any).

Purpose
To generate a password with a specific length and special characters:
```bash
node passwordGenerator.js --length 12 --specialChars
